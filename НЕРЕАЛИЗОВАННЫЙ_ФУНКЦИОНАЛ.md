## Полностью нереализованный функционал

### 1. КС-2, КС-3

Код рендереров заложен, но **отключён** в UI и API до появления официальных шаблонов ФНС.

### 2. ЭДО (реальный XML)

Интерфейс и поля БД (`edoStatus`, `edoXmlPath`) есть; сериализация `ON_NSCHFDOPPR` по XSD — не реализована.

### 3. АВР

Не реализован.

### 4. HTML-шаблоны (`template-engine.ts`)

Системные шаблоны в `system-templates.ts` не подключены к генерации — используется DOCX/XLSX-патчер.

### 5. SaaS (из ПРОДАКШЕН_ИДЕИ.md)

- Подписки / billing
- PLATFORM_MANAGER
- `mustChangePassword` при первом входе

---

## Частично реализованный функционал

### NumberingRule UI

Логика нумерации работает; отдельного UI настроек шаблона номера в `/settings` нет.

### Финансы

Экспорт, бюджет, счета/платежи в `finance/page.tsx`. Модель `Invoice` в БД не используется — счета идут через `Document` + `INVOICE`.

### Чат

- Текст, @упоминания, #проекты — ✅
- Вложения и emoji — ❌
